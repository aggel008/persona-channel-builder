# persona-channel-builder

OpenClaw skill — design and launch an autonomous AI-managed Telegram channel.

Interview → SOUL.md + CHANNEL.md + cron config + 3 sample posts. Ready to deploy.

## Install

```bash
clawhub install persona-channel-builder
```

## What it builds

| Output | What it is |
|--------|------------|
| `SOUL.md` | Who the persona is, how they think, what they notice |
| `CHANNEL.md` | Post formats, topic rules, what NOT to write |
| Cron config | Ready-to-paste OpenClaw job snippet |
| 3 sample posts | Voice validation before going live |

## How it works

4-block interview:

1. **The Channel** — topic, audience, language, posting frequency
2. **The Persona** — name, background, one specific real detail
3. **The Voice** — tone, what they never say, example post
4. **Infrastructure** — channel ID, timezone (optional)

At the end you have everything needed to deploy a channel that posts itself.
